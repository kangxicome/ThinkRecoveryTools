**ThinkRecoveryTools**

Some useful python scripts for Lenovo ThinkPad/Center recovery files.

| File                | Intro                                                        |
| ------------------- | ------------------------------------------------------------ |
| RecoveryUSBMaker.py | Make Recovery USB flash disk, needs files downloaded by Lenovo USBRecoveryCreator.exe . No more internet connections and account login needed. |
| RecoveryLite.py     | Move files in recovery disk created by USBRecoveryCreator.exe or RecoveryUSBMaker.py to archives folder, make your recovery OS much more cleaner. |
